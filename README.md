##description

VUE to XGMML converter, mainly used for exporting graphs from [Vue][1] to [Cytoscape 3][2].

##usage

```sh
./vue2xgmml.py input.vue output.xgmml
```

##requirements

- [NetworkX][3]
- PIL or [Pillow][4]
